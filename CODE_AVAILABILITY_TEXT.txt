Custom R scripts used to summarize candidate Fsite recurrence, branch-site positive-selection signals, candidate parallel substitutions, SSLS statistics, functional enrichment, gene-family copy number, genome architecture, transposable-element profiles, amino-acid centred log-ratio composition, salinity-transfer assays, public transcriptome summaries and ATG7 FoldX annotations are provided in Supplementary Software 1. The workflow takes standardized tabular outputs from BUSCO, OrthoMCL, IQ-TREE, PAML, RAxML, RepeatModeler2/RepeatMasker/LTR_retriever, DESeq2, ColabFold and FoldX, and writes analysis tables and Source Data files. The archived repository includes input schemas, configuration files, external command templates, method-code mapping and session-information output. The publication archive should include the final real input tables used for the analyses or a repository link to those tables.
