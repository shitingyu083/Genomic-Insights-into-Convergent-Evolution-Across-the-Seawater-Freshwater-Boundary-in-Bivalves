Custom R scripts used to summarize candidate Fsite recurrence, branch-site positive-selection signals, candidate parallel substitutions, SSLS statistics, functional enrichment, gene-family copy number, genome architecture, transposable-element profiles, amino-acid centred log-ratio composition, salinity-transfer assays, public transcriptome summaries and ATG7 FoldX annotations are provided in Supplementary Software 1. The workflow takes standardized tabular outputs from BUSCO, OrthoMCL, IQ-TREE, PAML, RAxML, RepeatModeler2/RepeatMasker/LTR_retriever, DESeq2, ColabFold and FoldX, and writes analysis tables and Source Data files. The archived repository includes input templates, configuration files, external command templates and session information.
